# Student Outreach & Campaign Analysis

## Project Overview
This project focuses on analyzing student outreach and campaign performance using Python, SQL, and Power BI. The objective was to evaluate campaign effectiveness, identify outreach trends, analyze applicant engagement, and build interactive dashboards for data-driven decision-making.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- SQL
- Power BI
- Jupyter Notebook
- Matplotlib

---

## Project Objectives
- Analyze applicant outreach performance
- Evaluate campaign effectiveness
- Identify response and conversion trends
- Perform exploratory data analysis (EDA)
- Build interactive dashboards
- Generate actionable business insights

---

## Dataset Information
The project uses outreach, campaign, and applicant datasets containing:
- Applicant details
- Campaign information
- Outreach activities
- Response status
- Conversion metrics
- Timeline-based engagement data

---

## Key Analysis Performed

### Outreach Data Analysis
- Duplicate Reference_ID detection
- Monthly application trends
- Campaign performance comparison
- Seasonal outreach analysis

### Applicant Data Analysis
- Duplicate applicant identification
- Country-wise applicant distribution
- Missing email analysis
- Data quality assessment

### Campaign Performance Analysis
- Outreach KPI tracking
- Conversion rate analysis
- Response rate monitoring
- Caller performance comparison

---

## Power BI Dashboard Features
The interactive dashboard includes:
- KPI Cards
- Conversion & Response Tracking
- Outreach Trend Analysis
- Campaign Performance Monitoring
- Monthly Application Trends
- Interactive Filters & Slicers

---

## Key Insights
- March and June showed the highest application activity.
- FA24IP and FA24SIC campaigns generated nearly half of all applications.
- Duplicate applicant and outreach records indicated data quality issues.
- Response and conversion trends helped identify campaign effectiveness.
