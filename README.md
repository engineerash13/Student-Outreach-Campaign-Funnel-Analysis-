## Dataset

The project uses outreach, campaign, and applicant datasets containing:

- Applicant details and country-wise distribution
- Campaign names and outreach activity records
- Response status and conversion metrics
- Timeline-based engagement data (monthly trends)
- Caller performance records

---

## Project Objectives

- Analyze applicant outreach performance across campaigns
- Evaluate campaign effectiveness and compare conversion rates
- Identify response trends and seasonal patterns
- Detect data quality issues (duplicates, missing values)
- Build interactive KPI dashboards for campaign monitoring

---

## Key Analysis Performed

**Outreach Data**
- Duplicate `Reference_ID` detection
- Monthly application volume trends
- Campaign performance comparison (FA24IP vs FA24SIC and others)
- Seasonal outreach pattern analysis

**Applicant Data**
- Country-wise applicant distribution
- Missing email record analysis
- Duplicate applicant identification

**Campaign Performance**
- Conversion rate and response rate tracking
- Caller performance comparison
- KPI development and monitoring

---

## Key Insights

- **March and June** showed the highest application activity across campaigns
- **FA24IP and FA24SIC** campaigns generated nearly half of all applications combined
- Duplicate applicant and outreach records flagged as data quality issues
- Response and conversion trends helped identify the most effective campaign strategies

---

## Power BI Dashboard Features

- KPI cards for total applicants, response rate, and conversion rate
- Monthly application trend line chart
- Campaign performance comparison visuals
- Caller effectiveness breakdown
- Interactive slicers for filtering by campaign, month, and status
